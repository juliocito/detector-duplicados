# 🧠 Detector de Archivos Duplicados

Herramienta en Python que detecta archivos duplicados dentro de una carpeta usando comparación por contenido (hash MD5).

---

## 🚀 Características

- Detecta archivos duplicados aunque tengan nombres diferentes
- Usa hashing (MD5) para comparar contenido
- Permite organizar o mover archivos duplicados
- Funciona con cualquier tipo de archivo

---

## 🛠️ Requisitos

- Python 3.x
